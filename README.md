# AS_Transciptomic_Analysis
Analysis of Ankylosing Spondylitis using public GEO datasets analyzed with the R programming language

# Project Overview
This study aims to identify significantly genes that may contribute to the molecular mechanisms underlying AS, especially using the R programming language

# Repository Structure
- data : top 500 genes exported from GEO2R
- results : filtered significant genes based on adj.p < 0.05 and  |log₂ fold change| ≥ 1
- figures : summary of the analysis flow
- report : final analysis report
